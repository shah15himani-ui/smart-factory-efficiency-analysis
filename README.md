# Manufacturing Process Health & Operational Efficiency Analysis

## Overview
This project analyzes smart factory machine data to evaluate:
- Machine health
- Production efficiency
- Defect rates
- Operational performance

## Key Features
- Machine Health Index calculation
- Cross-metric diagnostics
- Efficiency classification analysis
- Data-driven insights and recommendations

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Results
- Identified low-efficiency machines
- Found correlation between vibration and errors
- Detected inefficiencies in high-load operation mode
